# Day1: PRD — 찰리초이 자기소개 랜딩페이지

나의 홈페이지 - [charlychoi.menus.space](http://charlychoi.menus.space) 기반 

---

### 1. 목적 (Purpose)

AI 큐레이터·강사·작가 찰리초이의 전문성과 활동을 한눈에 보여주는 개인 브랜딩 랜딩페이지를 만든다. 처음 방문한 사람이 30초 안에 "이 사람이 누구이고 무엇을 하는지" 파악할 수 있어야 한다.

---

### 2. 사용자 (Users)

- 유튜브 구독자 (40~60대 중장년층)
- 강의 의뢰인 (기업·기관 담당자)
- 출판사·미디어 관계자
- 협업 제안자

---

### 3. 기능 및 구성 (Features)

**① 헤더 / 히어로 섹션**

- 이름: 찰리초이 (Charly Choi)
- 타이틀: AI 큐레이터 · AI 강사/AI 컨설턴트 · 작가
- 프로필 사진 (강의 현장 사진)
- 다크모드 토글 버튼
- 내비게이션: 소개 / 경력 / 전문분야 / 미디어 / 출판 / 강의 / 연락처

**② 소개 섹션 (#intro)**

- 한 줄 자기소개 문장
- 핵심 키워드 강조 (AI, 암호화폐, 중장년층 콘텐츠)

**③ 경력 섹션 (#career)**

- IT 컨설팅·교육 30년 경력 타임라인
- Google Workspace / Gemini 교육 담당
- 2024년 정년퇴직 후 유튜브·저술 활동 전환

**④ 전문분야 섹션 (#expertise)**

- AI 도구 활용 (Claude, Grok, Gemini 교차 검증)
- 암호화폐 규제 분석 (GENIUS Act, CLARITY Act, PACE Act 등)
- 시니어 AI 강사 및 AI 컨설턴트 (서울시 주관 프로그램)
- 콘텐츠 크리에이터 (유튜브 구독자 7,700명+)

**⑤ 미디어 섹션 (#media)**

- 유튜브 채널 링크: youtube.com/@CharlyChoi
- 채널 소개: 복잡한 경제·금융·AI 이슈를 쉽게 해설
- 구독자 수 뱃지 표시

**⑥ 출판 섹션 (#publications)**

- 저서 목록 카드 형태로 표시
    - ChatGPT 프롬프트 엔지니어링 마스터 (영어 KDP, 2024)

**⑦ 강의 섹션 (#teaching)**

- 서울시 시니어 AI 강사 및 AI 컨설턴트 활동
- Google Workspace / Gemini 기업 교육
- 강의 문의 버튼 → 연락처로 이동

**⑧ 연락처 섹션 (#contact)**

- 유튜브 채널 링크
- 이메일 또는 문의 폼
- SNS 링크 (해당 시)

---

### 4. 제약 조건 (Constraints)

- **언어:** 한국어 (일부 영문 병기 가능)
- **반응형:** 모바일 우선 설계 (40~60대 스마트폰 사용자 많음)
- **접근성:** 큰 글씨, 높은 명암 대비, 심플한 레이아웃
- **다크모드:** 토글 지원
- **기술 스택:** HTML + CSS + 간단한 JavaScript (Next.js 불필요, 순수 웹으로)
- **배포:** Vercel (GitHub 연동)
- **이미지:** 강의 현장 사진 1장 사용

---

### 5. 성공 기준 (Success Criteria)

- [ ]  처음 보는 사람이 30초 안에 찰리초이가 누구인지 파악 가능
- [ ]  스마트폰에서 깔끔하게 표시됨
- [ ]  유튜브 채널 링크 클릭 가능
- [ ]  저서 5권이 카드 형태로 표시됨
- [ ]  다크/라이트 모드 전환 작동
- [ ]  Vercel 인터넷 주소로 공개 접근 가능
